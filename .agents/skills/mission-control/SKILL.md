# Mission Control

System health monitoring for EWM Robotic Platform.

## When to Use
- Checking Docker container health and status
- Monitoring system resource utilization
- Investigating process failures
- Aggregating logs from multiple services

## Activation Commands
> "Run mission-control to check if all Docker containers are healthy"
> "Check the memory usage of Node-RED and Redis containers"
> "Show the last 50 log lines from sap-bridge service"
