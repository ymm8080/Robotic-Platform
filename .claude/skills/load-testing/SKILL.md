# Load Testing

System load testing for EWM Robotic Platform.

## When to Use
- Testing SAP bridge throughput under concurrent requests
- Validating MQTT message handling at scale
- Benchmarking robot dispatch latency under load
- Stress-testing Redis and PostgreSQL

## Activation Commands
> "Run load test on SAP bridge with 100 concurrent requests"
> "Measure MQTT message throughput with 50 AGVs reporting simultaneously"
> "Stress test the robot dispatch endpoint at 200 requests per second"
