# Hindsight Memory

Persistent knowledge management for AI agent teams.

## When to Use
- Storing project conventions and coding standards
- Recording architectural decisions and rationale
- Saving known pitfalls and workarounds
- Recalling past solutions to similar problems

## Activation Commands
> "Store the SAP RFC error handling convention in hindsight-memory for future reference"
> "Recall our previous decision on MQTT QoS levels for AGV communication"
> "Save the outbox pattern implementation notes to team memory"
