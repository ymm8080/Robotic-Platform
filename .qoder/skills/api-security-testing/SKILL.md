# API Security Testing

Security testing patterns for EWM Robotic Platform APIs.

## When to Use
- Testing SAP bridge API endpoints for vulnerabilities
- Validating JWT token handling and expiration
- Checking input sanitization on robot command endpoints
- Auditing authentication and authorization

## Activation Commands
> "Test the SAP bridge API for common security vulnerabilities"
> "Validate JWT token handling in the robot dispatch endpoint"
> "Audit API authentication for all exposed endpoints"
