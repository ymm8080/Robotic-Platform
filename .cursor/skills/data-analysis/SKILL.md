# Data Analysis

Analyze data for EWM Robotic Platform metrics and performance.

## When to Use
- Analyzing CSV/spreadsheet data of robot task completion times
- Finding bottlenecks in AGV route efficiency
- Creating visualizations of warehouse throughput
- Identifying patterns in error logs and failure rates

## Activation Commands
> "Analyze this CSV of robot task completion times and find slowest routes"
> "Create a visualization of hourly throughput for the last 30 days"
> "Find correlation between battery level and task failure rate"
