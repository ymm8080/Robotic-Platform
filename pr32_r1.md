## CRITICAL

### C1. MQTT Auth Removed - core/gateway.py
_load_mqtt_password() and MQTT_REQUIRE_AUTH deleted. Any MQTT client can publish VDA5050 orders. Violates Iron Rule #5.
Fix: Restore auth logic, use Docker Secrets.

### C2. /metrics Auth Contradicts PR - traffic_coordinator_main.py
PR says "/metrics unauthenticated for Prometheus" but code ADDS auth check. Prometheus scraping breaks with 401.
Fix: Remove _check_auth on /metrics.

### C3. /estop Validation Removed - traffic_coordinator_main.py
_SAFE_ID_RE validation on zone_id removed. Arbitrary strings accepted.
Fix: Restore validation.

## HIGH

### H1. SAP Memory Leak Cleanup Removed (Opposite of PR Desc) - sap_coordinator_bridge.py
PR says "cleanup logic to prevent memory leak" but code REMOVES cleanup. _confirmed/_submitted grow unboundedly.
Fix: Restore cleanup or use bounded structure.

### H2. AUTO_CONFIRM Default "0" -> "1" - sap_coordinator_bridge.py
Original warned "risky: cannot distinguish completed from failed". Failed tasks confirmed as completed in SAP.
Fix: Revert default to "0".

### H3. Lift Release on Dispatch Failure Removed - core/coordinator.py
_release_lifts_for_assignment() deleted. Lifts locked permanently on dispatch exception.
Fix: Restore lift release or reserve after successful dispatch.

### H4. _on_mqtt_inbound Exception Handling Removed - traffic_coordinator_main.py
Try/except removed. Any ingest_uplink exception crashes MQTT callback thread.
Fix: Restore try/except with logging.
